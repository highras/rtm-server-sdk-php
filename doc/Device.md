# Device Api

* `addDevice($uid, $appType, $deviceToken)`: Add device
    * `uid`: **(long)** User id
    * `appType`: **(strring)** app type, apns or fcm
    * `deviceToken`: **(strring)** deviceToken
    
* `removeDevice($uid, $deviceToken)`: Remove device
    * `uid`: **(long)** User id
    * `deviceToken`: **(strring)** deviceToken
